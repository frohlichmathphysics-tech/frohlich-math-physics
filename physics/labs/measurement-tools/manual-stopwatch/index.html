Manual Stopwatch — Test Page

physics/labs/measurement-tools/manual-stopwatch
What to test:
  • Drag the stopwatch body anywhere inside the dark frame.
  • Click Start. Hand stays at 12 (frozen). A small red dot pulses below center — that is the only visual cue that timing is in progress.
  • Click Stop after a few seconds. Hand should snap instantly to the elapsed-time position on the dial. Read the value to the nearest 0.1 s.
  • Click Reset. Hand returns to 12; dial is blank.
  • Try rapid Start / Stop / Reset cycles. State should remain consistent — Reset is allowed during any state (including while running, to abort a misclick).
  • Open this page on a phone. Drag and buttons should work via touch.
