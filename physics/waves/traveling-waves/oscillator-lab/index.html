Damping is shown as exponential decay in space — a simplification of real string damping. When you change the tension or linear density, the wave is reset (a real string would show a discontinuity propagating from the source). When the frequency knob is turned during a running continuous wave, the wavelength on the string updates everywhere instantly — phase at the source remains continuous.
